## Build and run

### To Run Locally:

Clone the repository and perform the following command line actions:

```bash
$> cd wordle-lwr
$> npm install
$> npm run start
```

#### To Develop:

```bash
$> cd wordle-lwr
$> npm install
$> npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in browser.

#### To Build:

```bash
$> yarn lwr build
```

This builds the static site in the `__generated_site__` folder.
